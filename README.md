# Arduino Wind
Library to easy the uploading of sensor data to [windguru](https://windguru.cz) website.

## Dependencies

- [HttpClient](https://github.com/arduino-libraries/ArduinoHttpClient)
- [Ethernet](https://github.com/arduino-libraries/Ethernet)
- [MD5Hash](https://github.com/tzikis/ArduinoMD5/)

It is strict necessary that you have already added to your Arduino IDE the libraries above.

## Instalation

Create a folder named _ArduinoWind_ in the _libraries_ folder inside your Arduino sketch folder. If the libraries folder doesn't exist, create it. Then copy everything inside. (re)launch the Arduino IDE.
